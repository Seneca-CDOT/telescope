Telescope Auth Service Test

This is a small client web app used to help test the auth service setup. It assumes you are running the microservices. If you haven't done so already, you should do the following in the root of the Telescope project: 

        pnpm run services:start

That will start the necessary services for this demo to work.

  • Start the login flow (Username=user1, Password=user1pass). Send state and get back state and a token.
  • Start the logout flow. Send state and expect to get the same state back.
